TxLINE Phantom Activation Helper

Connect Phantom, sign the TxLINE activation message, then activate the read-only API token. This page never asks for or exports your private key.

Before This Page

The first on-chain step is a TxLINE subscribe transaction. The helper below can create that transaction with Phantom, then use the returned transaction signature as txSig.

Free tiers from the TxLINE docs: service level 1 for World Cup and international friendlies with 60-second delay, or service level 12 for real time. The free tier charges no TxL tokens, but the Solana transaction still needs a tiny amount of SOL for network fees.
